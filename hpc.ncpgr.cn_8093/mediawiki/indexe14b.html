perl使用cpan管理perl的各种包，我们可以使用cpan来安装perl模块，避免使用源码安装遇到的各种问题。如果使用的perl，cpan安装的模块默认安装到了系统路径下，普通用户无此权限。我们需要将perl模块装到自己的帐号目录下，方法有很多，比如装一个自己的perl，不使用系统的，见Activeperl安装.此外，也可以通过cpanm来安装，这是一个比较简单的方法。

下载cpanm到自己的目录下

cpanm

更改权限

chmod 755 ~/software/perl/cpanm

将路径写到bashrc文件中

export PATH=~/software/perl/:$PATH

source一下bashrc文件

source ~/.bashrc

尝试安装一个模块

cpanm install SVG::GD

写入PERL5LIB变量

第一次安装时，cpanm会创建一个~/perl5目录，而perl模块的安装位置为~/perl5/lib/perl5，将这个变量写入bashrc文件的PERL5LIB变量中

export PERL5LIB=~/perl5/lib/perl5

source source ~/.bashrc

验证安装

perldoc SVG::GD

如果出现了SVG::GD的帮助文档 ，则安装成功。
