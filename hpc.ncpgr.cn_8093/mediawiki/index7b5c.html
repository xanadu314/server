Activeperl是ActiveState旗下的一个perl商业发行版，我们可以在ActiveState的官网上免费获取其最新的社区发行版。商业版和企业版需要付费。我们在自己的linux账户下配置自己的perl环境时一般选择这个发行版，而不是用perl官网的源码来编译，理由是Activeperl安装、配置比较简单，安装完成之后就可以用cpan来很方便地安装各种perl模块。

这里只讲其在linux下的安装,以Activeperl 5.18.4.1803版为例。

Activeperl下载

Activeperl下载，下载的时候注意软件版本，这里用的是64位linux版。

解压

tar -xf ActivePerl-5.18.4.1803-x86_64-linux-glibc-2.5-298573.tar.gz

安装

cd ./ActivePerl-5.18.4.1803-x86_64-linux-glibc-2.5-298573
sh ./install.sh

下面需要填yes or no的地方一律填yes。 安装的位置选择自己自home目录下的一个位置。 

Enter top level directory for install? [/opt/ActivePerl-5.18] ~/opt/ActivePerl-5.18

接下来又是一路的yes，安装过程时间略微有点长,最后出现如下字样表示安装完成。 

Thank you for using ActivePerl!

更改~/.bashrc配置文件

在~/.bashrc文件中添加这两行。 

export PATH=$HOME/opt/ActivePerl-5.18/bin:$PATH
export PERL5LIB=$HOME/opt/ActivePerl-5.18/lib:$PERL5LIB

然后 

source  ~/.bashrc

使~/.bashrc文件的修改生效。 查看一下安装是否成功which perl 

~/opt/ActivePerl-5.18/bin/perl

接下来你可以试一下使用cpan来安装模块了。
